SkillOpt: Executive Strategy for Self-Evolving Agent Skills

SkillOpt: 스스로 진화하는 에이전트 기술을 위한 실행 전략

Yifan Yang, Ziyang Gong, Weiquan Huang, Qihao Yang, Ziwei Zhou, Zisu Huang, Yan Li, Xuemei Gao 외 7명
  • 기존 에이전트 기술은 수작업, 일회성 생성, 또는 느슨한 자가 수정으로 개발되어 신뢰성 있는 개선이 어려웠습니다.
  • SkillOpt는 에이전트 기술을 고정된 에이전트의 외부 상태로 훈련하는 체계적인 텍스트 공간 최적화 방법입니다.
  • 별도의 최적화 모델이 점수화된 실행 결과를 바탕으로 기술 문서에 추가/삭제/교체 편집을 제안하고, 유효성 검사 점수가 향상될 때만 적용합니다.
  • SkillOpt는 6개 벤치마크, 7개 모델, 3개 실행 환경에서 모든 52개 평가 항목에서 최고 성능을 달성했으며, GPT-5.5에서 평균 정확도를 19.1~24.8점 향상시켰습니다.
  • 이 방법은 배포 시 추가 추론 호출 없이 안정적인 기술 훈련을 가능하게 하며, 최적화된 기술의 전이성도 입증했습니다.

TransitLM: A Large-Scale Dataset and Benchmark for Map-Free Transit Route Generation

TransitLM: 지도 없이 대중교통 경로를 생성하기 위한 대규모 데이터셋 및 벤치마크

Hanyu Guo, Jiedong Yang, Chao Chen, Longfei Xu, Kaikui Liu, Xiangxiang Chu
HF upvotes 169
  • 기존 대중교통 경로 계획은 지도 인프라와 복잡한 라우팅 엔진에 의존하며, 이를 우회하는 모델 학습용 데이터셋은 없었습니다.
  • 본 연구는 4개 중국 도시의 1,300만 개 이상 대중교통 경로 기록을 담은 대규모 데이터셋 TransitLM을 제안합니다.
  • TransitLM은 12만 845개 역과 1만 3,666개 노선을 포함하며, 연속 사전 학습 코퍼스 및 벤치마크로 활용됩니다.
  • 실험 결과, TransitLM으로 학습된 LLM은 높은 정확도로 유효한 경로를 생성하고 GPS 좌표를 역에 암묵적으로 연결합니다.
  • 이는 대중교통 경로 계획이 데이터만으로 학습 가능하며, 출발지-목적지 정보로부터 종단간(end-to-end) 지도 없는 경로 생성을 가능하게 함을 보여줍니다.
